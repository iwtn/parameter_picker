# URL Rearrange
Split and combine URLs as you like.

## Development

build with webpack.

```bash
$ npx webpack
```

## Install
https://chrome.google.com/webstore/detail/url-rearrange/flnmdckhlhmejifondmiclogolepknfd

## Features
- Copy the divided and combined URL to the clipboard as you like.
